Wireframe

When sketching out my plan for the layout for the website I wanted it to be quite simple in order to make it easy for the user to navigate. I included two links to each page. The first in the header, which remains consistent on all pages, to allow the user to easily swap through the different categories. The second is on the homepage, where there are boxes with matching titles to the other buttons but these ones include a subtitle which has the full name of each activity (e.g. CARP has Contrast, Alignment, Repetition and Proximity). These subtitles ensure that the user knows what is on each page without clicking on it.
